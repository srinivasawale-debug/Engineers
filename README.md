# RozgarSaathi

## Problem Statement

Daily wage workers such as plumbers, electricians, painters, and mazdoors often wait at local job points (nakas) without guaranteed work. They lack digital visibility and access to nearby job opportunities.

## Solution

RozgarSaathi is a hyperlocal platform where workers can register their skills and availability, and households or contractors can post same-day jobs nearby. The platform directly connects workers and employers without any middlemen or commission.

## Key Features

* Worker registration with skill tags
* Job posting by households or contractors
* Location-based job matching
* Same-day job availability
* Simple worker contact system

## Tech Stack

Frontend: React / HTML / CSS
Backend: Node.js / Express
Database: MongoDB
Mobile App: React Native or Flutter

## Project Structure

/frontend – UI components
/backend – APIs and server logic
/database – database models
/docs – project documentation

## Future Improvements

* GPS-based worker discovery
* Worker rating system
* In-app chat between worker and employer
* Push notifications for job alerts
